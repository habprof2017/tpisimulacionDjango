Introduccion

  • En la parte superior los links "Lote 1" y "Lote 2" son las pantallas para la generacion de las variables de cada lote respectivamente
  • La opcion de "Variables Generadas" permite ver las variables que se generaron
  • Las variables generadas se guardan en una base de datos SQLite 3

  • Se utilizo el lenguaje Python 3.5 y el Framework Web Django (ver 1.11)
  • El proyecto esta alojado en el hosting de "Python Anywhere"con una cuenta gratuita.
  • De esta manera se evitan los instaladores (y la guia para la instalacion), instalacion de librerias externas y es independiente del SO que se utiliza
